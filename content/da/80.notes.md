## Create note bobbles

Double click anywhere in the document to create a note bobble.
![Note Bobble](/images/da/notes/note-bobble.jpg "Double Click to create the movable note bobble"){.w-[84px] .inline-block}

The nearby text is automatically selected and inserted inserted into the bobble.

### Move note bobble

The note bobble is movable. After moving it stays where you moved it forever (if you don't move it again).

### Edit the note

Click on the note bobble to edit the text or add content.
![Note Text Box](/images/da/notes/note-text-box.jpg "Click on the note bobble or the highlighted text to edit or share it."){.w-1/2 .inline-block style="float: right; margin-left: 1rem;"}

The quotation of the text on the page, have to be 100% identical. If not the share function will not work (at all).

## Highlight text

Hold a word to begin the highlight process.
![Highlighted text](/images/da/notes/highlight-text.jpg "Hold a word to highlight it."){.w-[96px] .inline-block}

### Edit highlighted text

The length of the highlighted area isn't editable, and it's not advisable to add text (in the highlighted text-box) if you aren't planning to print/email etc. your notes and use it in your preach, lecture or teaching.

## Share

The text between the france «apostrophs» is used when you want to share the note or the highlighted text. This text have to be 100% identical with the original, if not no sharing.

NB! Open the note to share it.
![Note Menu Alternatives](/images/da/notes/menu-alternatives.jpg "Open the note bobble or highlighted text to get the share option"){.w-1/2 style="float: right; margin-left: 1rem;"}

## Link to notes

In the overview of notes you can click on the note to open the page with the note.

The note app will automatically scroll down to the note.

## Note Actions

In the footer of the Note Menu (cp) there is an Action Menu with all available actions:
![Note Command Palette](/images/da/notes/note-action-menu.jpg "Note Command Palette Menu"){.w-1/2 style="float: right; margin-left: 1rem;"}

### Print & export etc

Copy, Print, Email, Export and/ or save your study notes.

> If you are a student, priest, pastor or teacher it is advisable to add comments in the note text boxes, then you will have a very good summary of the sermon(s) in the "printed" version of your notes.

### Import

If you don't find your notes next time your are visiting the web site you can import the notes and continue where you ended.

> NB! You should only import your notes to the device where you was creating them.
> If different screen width, the notes will be misplaced.
